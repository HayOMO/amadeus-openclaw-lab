# Persona Profile Source Notes

This file records why selectable persona packs were added. Runtime injection
uses only the files under `persona/profiles/*`; these notes are maintenance
context and are not injected into model prompts.

## 2026-06-27 Additions

- `classic_nekomimi`: added as an original classic catgirl archetype. It is not
  based on a specific copyrighted character or third-party card; the pack is
  designed around useful observant/cute assistant behavior with restrained
  catlike flavor.
- `hatsune_miku`: added for a bright collaborative creator/virtual-singer
  voice. Crypton official profile and Piapro context were used for factual
  grounding; public card repositories showed sustained Miku card demand.
- `raiden_shogun_ei`: added for a formal dual-aspect authority/reflection
  voice. Genshin profile material and public card pages both emphasize the
  Shogun/Ei split, eternity, and Inazuma framing.
- `aqua_konosuba`: added for a high-energy comedy support voice. KonoSuba
  references ground her as water goddess / Arch Priest; the card keeps the
  useful purification/support angle instead of only noisy chaos.

Selection rule: prefer non-male profiles with distinct speaking modes and
non-overlapping tone. For copyrighted characters, use enough public reference
material to write a compact card plus lorebook without copying third-party
roleplay cards. For original archetypes, write the behavior target directly and
keep the card free of runtime/tool identity.
