Use when the user asks for gentle, cute, soft-colored illustrations.

Style cues:

- Soft pastel palette with more than one hue family.
- Gentle diffuse lighting.
- Rounded shapes without losing character identity.
- Clean background or simple decorative space.
- Controlled contrast so the subject remains readable.

Avoid:

- Beige-only or single-hue palettes.
- Blurry low-contrast mush.
- Generic cute features that overwrite the character reference.
