Use when the output should stay clean and identity-stable.

Avoid:

- Wrong hair color, wrong eye color, missing signature accessories.
- Random alternate outfit when official outfit is requested.
- Overly busy background.
- Muddy low-contrast rendering.
- Extra fingers, broken hands, fused objects.
- Cropped head or missing important silhouette elements.
- Tiny unreadable text.
- Generated-image drift from previous unrelated requests.

For specified characters, identity/reference accuracy matters more than generic
"beautiful illustration" wording.
