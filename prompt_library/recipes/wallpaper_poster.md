Use for wallpapers, posters, covers, banners, and visually polished hero images.

Prompt shape:

- Main subject and pose.
- Orientation and framing: portrait, landscape, phone wallpaper, desktop
  wallpaper, poster.
- Foreground/midground/background relationship.
- Lighting direction and color contrast.
- Negative space for icons/text if the user needs it.
- Clean identity/reference constraints when a specific character is involved.

Avoid:

- Vague "cool background" wording.
- Too many unrelated props.
- Cropping important character details unless intentionally close-up.
- One-color palettes unless the user asks for monochrome.
