Use for Chinese social-media covers, 小红书 notes, 抖音/公众号 covers,
knowledge cards, product seeding cards, and tutorial thumbnails.

Prompt shape:

- Platform/use: 小红书封面, 笔记配图, 抖音封面, 公众号头图, 商品种草卡,
  课程封面, 穿搭诊断图, 产品一生科普信息图.
- Layout first: vertical 3:4 or 4:5, strong title area, one hero object/person,
  3-5 content blocks, label strips, arrows/callouts, card grid, or ticket layout.
- Chinese typography: large bold Chinese title, short subtitle, small labels only
  when needed. Keep generated text short and inspect it.
- Visual flavor: clean editorial card, soft tech card, modern business card,
  black-white grid, ticket/pass layout, neo-constructivist teaching card, or
  lifestyle collage.
- Attention hooks: contrast, clear hierarchy, useful blank space, one theme
  color plus one accent color, not random sticker clutter.

Good patterns:

- "产品/食物的一生": one object shown through stages, direct labels, simple
  background, vertical card.
- "穿搭/妆造诊断": left hero portrait/outfit, right structured analysis blocks.
- "教程/清单封面": title top, key number large, 3-5 tiles, clear visual rhythm.

Avoid:

- Western SaaS hero-page look unless requested.
- Dense English-only typography.
- Too many text blocks; generate background/structure first if final text must
  be edited manually.
- Beige-only lifestyle boards with no contrast.
