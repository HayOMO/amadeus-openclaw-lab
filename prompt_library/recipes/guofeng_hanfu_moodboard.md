Use for 国风/汉服/妆造/古风 visual proposals, Chinese fashion moodboards,
改良唐风/宋制/明制 inspiration boards, and brand/editorial boards.

Prompt shape:

- Board format: horizontal 4:3 or vertical 3:4 proposal board; left hero visual,
  right modular design details.
- Hero: Chinese model or illustrated character wearing the requested hanfu or
  modernized Chinese-inspired outfit; specify dynasty inspiration only if known.
- Modules: clothing silhouette, hair design, makeup, jewelry, fabric/material,
  color palette, motif, accessory close-up.
- Aesthetic lane: 清冷国风, 新中式, 唐风甜美, 宋制温婉, 水墨留白,
  国潮商业海报, 古典妆造提案.
- Palette: name 2-4 colors and one material cue, such as jade green, cinnabar,
  ivory silk, ink black, warm gold embroidery.

Avoid:

- Mixing unrelated dynasties when the user asks for historical accuracy.
- Generic "oriental fantasy" wording.
- Overusing lanterns, dragons, and random calligraphy.
- Treating generated Chinese text as final without inspection.
