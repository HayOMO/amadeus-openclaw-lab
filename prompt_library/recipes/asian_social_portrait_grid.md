Use for 小红书/日系/韩系 portrait sets, profile photos, fashion posts,
four-grid/nine-grid photos, and realistic social-media portrait aesthetics.

Prompt shape:

- Input rule: if the user provides photos, preserve identity, face shape,
  hairstyle, and recognizable features. Only change pose, outfit, lighting, or
  mood as requested.
- Grid/layout: 2x2 四宫格, 3x3 九宫格, contact sheet, magazine contact sheet,
  each panel with a different pose/expression but same person.
- Style lane: 日系清透, 韩系甜酷, 港风胶片, 法式复古, 国风妆造, 校园写真,
  街拍, 杂志内页, 证件感精修.
- Details: outfit texture, hair styling, makeup, small prop, season, location,
  natural skin texture, realistic light.
- Camera look: soft window light, warm film grain, clean studio light, outdoor
  overcast, night neon, shallow depth of field.

Avoid:

- Western influencer glam as the default.
- Over-sculpted facial features, plastic skin, heavy contour, random ethnicity
  drift.
- A single repeated expression across all grid panels.
- Changing the person's identity when the request is a portrait edit.
